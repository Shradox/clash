# ARENA WARS — Мультиплеер

## Быстрый старт

### 1. Установка
```bash
cd arena-wars
npm install
```

### 2. Запуск сервера
```bash
node server.js
```

### 3. Открыть в браузере
```
http://localhost:3000
```

Открой в **двух вкладках** или на **двух устройствах** в одной сети.

---

## Деплой в интернет (Render.com — бесплатно)

1. Загрузи папку `arena-wars` на GitHub
2. Зайди на https://render.com → New Web Service
3. Выбери репозиторий
4. Build Command: `npm install`
5. Start Command: `node server.js`
6. Готово! Получишь URL вида `https://arena-wars.onrender.com`

---

## Архитектура (авторитетный сервер)

```
Клиент A          Сервер (Node.js)         Клиент B
   |                    |                      |
   |-- place card ----> |                      |
   |                    |-- tick игры 20/с --> |
   |                    |   (вся логика)        |
   | <-- snapshot --    |    <-- snapshot --    |
   |   (x,y,hp,...)     |    (x,y,hp,...)      |
```

- Клиент **только рендерит** — никакой игровой логики
- Сервер **авторитетен** — нельзя читерить
- Синхронизация: снэпшот 20 раз/сек (50ms)
- Интерполяция позиций на клиенте для плавности
